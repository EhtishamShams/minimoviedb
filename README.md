# MiniMovieDB

A minimilist replica of [TheMovieDB](https://www.themoviedb.org) for Commaful code screening. Contains the layout for the following page:
https://www.themoviedb.org/movie/283995-guardians-of-the-galaxy-2

## Project setup

```
npm install
```
